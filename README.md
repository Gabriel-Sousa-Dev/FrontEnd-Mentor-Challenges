### O que eu aprendi 

Eu aprendi a utilizar Variáveis css e a utilizar o padrão Bem Css.

```css
:root{
  --White: hsl(0, 0%, 100%);
  --LightGray: hsl(212, 45%, 89%);
  --GrayishBlue: hsl(220, 15%, 55%);
  --DarkBlue: hsl(218, 44%, 22%);

```
